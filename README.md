## 📈 Overview

**PGC Backtest** is a modular, It is a powerful Python-based backtesting framework tailored for intraday, weekly, and monthly options strategies. It supports advanced features like MTM tracking, SL/TP logic, premium decay modeling, dynamic strike selection (straddles/strangles), and Telegram alerts. Built for high-frequency data analysis using per-minute OHLC and option chain data, this tool enables systematic strategy evaluation at scale.
